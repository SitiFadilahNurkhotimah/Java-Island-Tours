# Java Island Tours: Sistem Manajemen Basis Data Relasional Berbasis Web

Telah mengembangkan situs _travel agency_ yang fungsional menggunakan PHP dan VS Code, yang terintegrasi dengan basis data MySQL yang dikelola melalui XAMPP dan phpMyAdmin. Proyek ini berfokus pada penerapan arsitektur basis data terstruktur dengan tabel-tabel yang dinormalisasi serta hubungan kunci utama dan kunci asing untuk memastikan integritas data. Sistem ini mendukung operasi CRUD yang lancar untuk mengelola paket wisata, pemesanan pelanggan, dan catatan perjalanan secara efisien.
